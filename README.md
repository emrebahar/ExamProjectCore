# ExamProjectCore
(.Net Core Project)  Kullanıcı adı ve şifre ile panele giriş yapılması. "wired.com" da yayınlanan son 5 yazıyı sayfa içine otomatik olarak alınması. Alınan bu yazılardan biri seçtirilerek yazı hakkında 4 adet soru hazırlanıp sisteme kaydedilmesi. Kaydedilen sınavı kullanıcının karşısına çıkartıp, verdiği cevaplara göre doğru ve yanlış cevaplarının gösterilmesi.
